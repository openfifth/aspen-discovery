## Aspen Discovery Updates
//mark n
### Other Updates
- Automatically insert translation terms in English when Google Translate is active when they are first viewed. (DIS-1700) (*MDN*)
- When using the backupAspen script, export rows in chunks to not exceed max packet sizes on restore. (DIS-1701) (*MDN*)

//kirstien
### API Updates
- In Search API, added getBrowseCategories API endpoint to retrieve a list of browse categories and related record results, also includes sub-categories if applicable. (DIS-1447) (*KK*)
- In Search API, updated getSubCategories endpoint to include different formatting if LiDA request. (DIS-1447) (*KK*)
- In User API, updated updateBrowseCategoryStatus to handle hiding all subcategories for a given category if hide parameter is set to 'all.' (DIS-1447) (*KK*)
- In Search API, marked getAppActiveBrowseCategories as deprecated to be removed in a future release. (DIS-1447) (*KK*)

//kodi

//leo
### E-Commerce Updates
- Stripe payment receipts are automatically captured and stored in Payment History and can be viewed after a successful payment.
- Payment information in Stripe now includes a description and detailed metadata, such as transaction type, patron ILS ID, patron barcode, fine count, and fine IDs for better reporting.
- Stripe payments now respect the system's configured currency code and symbol, not just USD.
