### Hero Slider Updates
- Introduced a comprehensive Hero Slider and Digital Signage system for creating dynamic image carousels and full-screen displays. ([DIS-1416](https://aspen-discovery.atlassian.net/browse/DIS-1416)) (*LS*)
  - Add Hero Slider Locations to define display areas with configurable aspect ratios and display styles.
  - Add Hero Slider Playlists to organize collections of images with customizable display duration per slide.
  - Extended Image Upload to support hero slider images with automatic aspect ratio calculation, alt text, page links, and date-based visibility controls.
  - Integrate hero sliders as a Portal Cell source type in Web Builder for embedding into pages.
  - Support per-slide duration control allowing different display times for each image in a playlist.
  - Add aspect ratio validation to warn administrators when playlists contain images with different aspect ratios.
  - Website Display Style implements a carousel using SwiperJS with smooth slide transitions, auto-hiding navigation controls that reveal on hover/touch/focus, Pause/Play button, screen reader accessibility, and adaptive contrast controls with semi-transparent backgrounds and backdrop blur.
  - Digital Signage Display Style provides full-screen image rotation with automatic progression, seamless content refresh by prefetching updates during the last slide, and XOGO digital signage system integration with auto-refresh URL parameter.
- Add soft-deletion support for Hero Slider Locations and Playlists. ([DIS-1416](https://aspen-discovery.atlassian.net/browse/DIS-1416)) (*LS*)

<div markdown="1" class="settings">

#### New Permissions
- Administer All Hero Sliders - Allows the user to manage hero sliders for all libraries.
- Administer Library Hero Sliders - Allows the user to manage hero sliders for their home library.

#### New Permission Group
- Administer Hero Sliders - Specify whether the role can manage all hero sliders or only those for the user's home library.

#### New Settings
- Local Catalog Enrichment > Hero Slider Locations
- Local Catalog Enrichment > Hero Slider Playlists

</div>
