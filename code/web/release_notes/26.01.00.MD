### E-Commerce Updates
- Stripe payment receipts are automatically captured and stored in Payment History and can be viewed after a successful payment.
- Payment information in Stripe now includes a description and detailed metadata, such as transaction type, patron ILS ID, patron barcode, fine count, and fine IDs for better reporting.
- Stripe payments now respect the system's configured currency code and symbol, not just USD.
