## Aspen Progressive Web App(PWA) Updates
- Added Progressive Web App option called Aspen Progressive Web App(PWA) ((DIS-1404)[https://aspen-discovery.atlassian.net/browse/DIS-1404])(*IT*)
- Added Permissions for managing Aspen Progressive Web App(PWA) ((DIS-1404)[https://aspen-discovery.atlassian.net/browse/DIS-1404])(*IT*)
- Added Service Worker to handle Notifications from within application for Aspen Progressive Web App(PWA) ((DIS-1404)[https://aspen-discovery.atlassian.net/browse/DIS-1404])(*IT*)
