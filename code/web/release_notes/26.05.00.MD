#### Event Registration
This is the first submission in a serie of enhancements adding a registation system for Aspen Native Events. ([DIS-1597](https://aspen-discovery.atlassian.net/browse/DIS-1597)) (*JOM* & *CZ*)
 - Administrators can enable registration at the event level. ([DIS-1597](https://aspen-discovery.atlassian.net/browse/DIS-1597)) (*CZ*)
 - Logged in patrons can register to specific event instances. ([DIS-1597](https://aspen-discovery.atlassian.net/browse/DIS-1597)) (*JOM* & *CZ*)
    - directly from the search results. 
    - from an event's detail page.
    - or from their list of saved event instances.
    - visitors who are not logged in will be presented with the option to do so upon opening the registration modal.
 - Patrons who have registered to an event can unregister. ([DIS-1597](https://aspen-discovery.atlassian.net/browse/DIS-1597)) (*CZ*)
    - directly from the search results.
    - from an event's detail page.
    - or from their list of saved event instances.
 - This includes linked account support, which means: ([DIS-1597](https://aspen-discovery.atlassian.net/browse/DIS-1597)) (*JOM* & *CZ*)
    - patrons with linked users can register their linked users to and ungregister them from events.
    - patrons with linked users can see whether their linked users are or are not registered to an event.
 - Event seats
    - Administrators can decide on a set number of seats per events. ([DIS-1597](https://aspen-discovery.atlassian.net/browse/DIS-1597)) (*JOM* & *CZ*)
    - The number of seats available can be overriden for a specific date / time (a specific instance of an event). ([DIS-1597](https://aspen-discovery.atlassian.net/browse/DIS-1597)) (*JOM*)
    - Seat availability is displayed ([DIS-1597](https://aspen-discovery.atlassian.net/browse/DIS-1597)) (*JOM*)
        - directly from the search results.
        - from an event's detail page.
        - or from their list of saved event instances.
