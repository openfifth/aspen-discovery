## Aspen Discovery Updates
// mark
### API Updates
- Remove unused AnodeAPI. (DIS-1372) (*MDN*)

### Grouped Work Display Updates
- In new horizontal format display, show local copies only when showing copies for a format. (DIS-1366) (*MDN*)
- Correct display of MPAA Rating in record details. (DIS-1370) (*MDN*)

### Hoopla Updates
- Add Record Extraction Batch Size for Hoopla to help prevent timeouts extracting content. (DIS-1369) (*MDN*)
- Reorganize Hoopla Settings. (DIS-1369) (*MDN*)

<div markdown="1" class="settings">

#### New Settings
- Hoopla > Settings > General Indexing Settings > Record Extraction Batch Size

</div>

### Indexing Updates
- Add options to index 896-899 fields as series (default on). Sideloads will not index these optional fields. 
- Cleanup solr index to not store fields that are not used within PHP. (DIS-1370) (*MDN*)

<div markdown="1" class="settings">

#### New Settings
- ILS Integration > Indexing Profiles > Indexing Options > Series > Index 896 as Series
- ILS Integration > Indexing Profiles > Indexing Options > Series > Index 897 as Series
- ILS Integration > Indexing Profiles > Indexing Options > Series > Index 898 as Series
- ILS Integration > Indexing Profiles > Indexing Options > Series > Index 899 as Series

</div>

### Item Display
- Update periodicals to be sorted using date parsing of call numbers to sort periodicals in reverse order. Records are sorted in Where Is It popup, Item Display, default items shown for a format. (DIS-67) (*MDN*)
- Properly sort records by nearby locations within Where Is It popup. (DIS-30) (*MDN*)
- Code cleanup when loading records and items from the database. (DIS-67) (*MDN*)
- Remove old unused code when items were stored in Solr and not the database. (DIS-67) (*MDN*)

### Polaris Updates
- Automatically deactivate any pickup areas that no longer exist within Polaris. (DIS-1326) (*MDN*) 

### Series Updates
- When the series module is active, do not remove series that are wholly contained within another series name. (DIS-1346) (*MDN*)
- Correct broken series image when viewing a series home page when the series has not been indexed yet. (DIS-1347) (*MDN*)

### User List Updates
- Correct generating covers for a list if the cover for a list entry cannot be loaded either because it doesn't exist or because it is invalid. (DIS-1315) (*MDN*)

### Other Updates
- Fix notice when no results are found for a search and Talpa is not enabled. (DIS-1316) (*MDN*)
- Improve the performance of the backupAspen script to execute quicker and take less memory. (DIS-1375) (*MDN*)
