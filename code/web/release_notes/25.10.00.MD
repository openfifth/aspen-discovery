## Aspen Discovery Updates
// mark
### Account Updates
- Correctly show recaptcha error message if recaptcha validation fails during self registration. (DIS-1384) (*MDN*)

### API Updates
- Remove unused AnodeAPI. (DIS-1372) (*MDN*)

### Grouped Work Display Updates
- In new horizontal format display, show local copies only when showing copies for a format. (DIS-1366) (*MDN*)
- Properly display MPAA Ratings for grouped works and MARC views. (DIS-1388) (*MDN*)
- Add the ability to sort eContent sources within a grouped work. (DIS-1380) (*MDN*)
  - The order of eContent can be customized by name of eContent Source. 
  - There is an option to sort available sources first. 
  - If the order of an eContent Source is not specified, it will be sorted alphabetically after sources that are explicitly specified.

<div markdown="1" class="settings">

#### New Settings
- Catalog / Grouped Works > eContent Sorting
- Catalog / Grouped Works > Grouped Work Display > Format and Edition Display > eContent Sorting

#### New Permissions
- Grouped Work Display > Administer All eContent Sorting - Allows users to change how eContent Sources are sorted within a grouped work for all libraries.
- Grouped Work Display > Administer Library eContent Sorting - Allows users to change how eContent Sources are sorted within a grouped work for their library.
</div>

### Hoopla Updates
- Add Record Extraction Batch Size for Hoopla to help prevent timeouts extracting content. (DIS-1369) (*MDN*)
- Reorganize Hoopla Settings. (DIS-1369) (*MDN*)

<div markdown="1" class="settings">

#### New Settings
- Hoopla > Settings > General Indexing Settings > Record Extraction Batch Size

</div>

### Indexing Updates
- Add options to index 896-899 fields as series (default on). Sideloads will not index these optional fields. 

<div markdown="1" class="settings">

#### New Settings
- ILS Integration > Indexing Profiles > Indexing Options > Series > Index 896 as Series
- ILS Integration > Indexing Profiles > Indexing Options > Series > Index 897 as Series
- ILS Integration > Indexing Profiles > Indexing Options > Series > Index 898 as Series
- ILS Integration > Indexing Profiles > Indexing Options > Series > Index 899 as Series

</div>

### Item Display
- Update periodicals to be sorted using date parsing of call numbers to sort periodicals in reverse order. Records are sorted in Where Is It popup, Item Display, default items shown for a format. (DIS-67) (*MDN*)
- Properly sort records by nearby locations within Where Is It popup. (DIS-30) (*MDN*)
- Code cleanup when loading records and items from the database. (DIS-67) (*MDN*)
- Remove old unused code when items were stored in Solr and not the database. (DIS-67) (*MDN*)

### Polaris Updates
- Automatically deactivate any pickup areas that no longer exist within Polaris. (DIS-1326) (*MDN*) 

### Search Updates
- Correct display of the Collection facet. (DIS-1391) (*MDN*)

### Series Updates
- Add a default sort method for series (defaults to volume). That sort method is used on public facing series page. (DIS-1394) (*MDN*)
  - Valid sorts are: Volume, Title, Publication Date, Publication Date Descending, Custom
- When the series module is active, do not remove series that are wholly contained within another series name. (DIS-1346) (*MDN*)
- Correct broken series image when viewing a series home page when the series has not been indexed yet. (DIS-1347) (*MDN*)

### User List Updates
- Correct generating covers for a list if the cover for a list entry cannot be loaded either because it doesn't exist or because it is invalid. (DIS-1315) (*MDN*)

### Other Updates
- Fix notice when no results are found for a search and Talpa is not enabled. (DIS-1316) (*MDN*)
- Improve the performance of the backupAspen script to execute quicker and take less memory. (DIS-1375) (*MDN*)
- Handle additional cases that cause errors when importing backups using the importAspenBackup script. (DIS-1386) (*MDN*)
- Update timestamp format when logging in Check Solr For Deleted Works Cron. (DIS-1389) (*MDN*)

### Cover Images Updates
- Added Bengali script support for the generation of default covers. (DIS-1355) (*LS*)
- When "Use Original Cover URLs" is enabled, standardized cover images dimensions, prevented relatively small images from displaying, and modified how they are stored. (DIS-1396) (*LS*)

### Digital Archives Updates
- Fixed Open Archives indexer to prevent image URLs (JPG, PNG, MP3, PDF) from being selected as record identifiers when they appear anywhere in the URL path. (DIS-1387) (*LS*)
- Fixed Open Archives cover images not displaying when URLs contain HTML entities. (DIS-1387) (*LS*)
- Added support for Omeka installation thumbnail extraction with new regex pattern for `<img class="full"...>` elements. (DIS-1387) (*LS*)

### Holds Updates
- Corrected an issue where a locally owned and available item at a search location would incorrectly display as "Checked Out/Available Elsewhere". (DIS-1297) (*LS*)

### Koha Updates
- Fixed an issue where a password reset via Aspen to Koha would succeed, but an error message would display instead. (DIS-1383) (*LS*)
