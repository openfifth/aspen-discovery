### Usage Updates
#### Usage Graph Reporting Periods
- The General Aspen Usage, API Usage, Sideloads Usage, Summon Usage, Materials Request Usage, and Axis360 Usage graphs can now be displayed and downloaded grouped by a selectable reporting period. ([DIS-1689](https://aspen-discovery.atlassian.net/browse/DIS-1689)) (*CZ*)
    - "Month" remains available and is the default, preserving the existing behaviour.
    - "Day" and "Year" group the same metrics by the respective period.
    - "Custom" groups usage into fixed-length periods defined by a start date and a duration in days.
- The usage data underlying these graphs now records the day, in addition to the month and year, so day-level reporting is possible. ([DIS-1689](https://aspen-discovery.atlassian.net/browse/DIS-1689)) (*CZ*)
    - usage recorded before this change is attributed to day 0 of its month so that historical data remains viewable under the new periods.
- Dates shown on the usage graphs and in their CSV downloads are formatted according to the configured locale. ([DIS-1689](https://aspen-discovery.atlassian.net/browse/DIS-1689)) (*CZ*)
