### User List Updates
- Add Facets to User Lists. ([DIS-160](https://aspen-discovery.atlassian.net/browse/DIS-160)) (*MDN*)
  - User Lists can now be filtered by the content in the list (Library Materials, Events, Series, Lists, etc).
  - When User Lists are only contain Library Materials (Grouped Works) or are filtered to only show Grouped Works, the same facets that appear in normal Library Catalog Searches will be active. 
  - Breadcrumbs for a list are updated to show the current range of filtered entries being shown as well as a count of the number of unfiltered entries.
  - When Grouped Works are indexed, information about the lists they are contained on will be indexed within Solr. 
  - If all entries on a list are not shown, the user will not be able to delete all entries, create a Collection Spotlight, or add the List to the Home page.
  - If all entries on a list are not shown, the user will not be able to reorder titles when using user defined order.
  - When one or more filters are applied, printing, emailing, exporting to CSV, exporting to RIS, and generating Citations will all reflect the filtered list.
  - When updating a list entry for a grouped work, the work will be reindexed automatically.
  - Remove redundant Availability and Number of Copies sorts for Lists. These did the same sorting. The combined sort has been renamed to Available Copies.
  - Previous functionality to filter only by format has been removed.
  - After update, all titles that are on a list will be reindexed. 
- Actions within the User List Toolbar have been consolidated for better usability. ([DIS-160](https://aspen-discovery.atlassian.net/browse/DIS-160)) (*MDN*)
- Ensure all content types show Edit form properly. ([DIS-160](https://aspen-discovery.atlassian.net/browse/DIS-160)) (*MDN*)
