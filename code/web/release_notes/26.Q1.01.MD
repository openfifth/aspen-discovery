# Quarterly Hotfix 1

---
The following notes include hotfix changes that were released on 02/11/26.

## Aspen Discovery Updates

### Account Updates
- Show count of titles Ready For Pickup from eContent Sources on Account Home page. ([DIS-1845](https://aspen-discovery.atlassian.net/browse/DIS-1845)) (*MDN*)

### Administration Updates
- Remove save buttons for locked records the administrator cannot edit. ([DIS-1817](https://aspen-discovery.atlassian.net/browse/DIS-1817)) (*MDN*)
- Ensure individual objects cannot be deleted if deletes are prevented for the object type. ([DIS-1818](https://aspen-discovery.atlassian.net/browse/DIS-1818)) (*MDN*)

### API Updates
- Fix loading list entries for some Browse Categories based on List default sort. ([DIS-1852](https://aspen-discovery.atlassian.net/browse/DIS-1852)) (*MDN*)

### CloudLibrary updates
- Only show cancel button if hold is not available because available holds will fail to cancel. ([DIS-1854](https://aspen-discovery.atlassian.net/browse/DIS-1854)) (*IT*) 

### Indexing Updates
- Protect against null pointer error if a shelf location is not mapped. ([DIS-1846](https://aspen-discovery.atlassian.net/browse/DIS-1846)) (*MDN*)
- Properly handle ISBNs with additional text following the ISBN. ([DIS-1813](https://aspen-discovery.atlassian.net/browse/DIS-1813)) (*MDN*)

### Hold Updates
- Automatically load list of users to use when placing a hold if "Hide Pickup Location Prompt" is on. ([DIS-1724](https://aspen-discovery.atlassian.net/browse/DIS-1724)) (*MDN*) 

### Hoopla Updates
- Fix Hoopla v2 so instant titles no longer duplicate once per member library; instant items now remain a single shared record with all applicable library scopes.([DIS-1812](https://aspen-discovery.atlassian.net/browse/DIS-1812)) (*YL*)
- Prevent Hoopla v2 updates from timing out. ([DIS-1821](https://aspen-discovery.atlassian.net/browse/DIS-1821)) (*MDN*)
- Update Hoopla checkout error handling to display clearer error messages to patrons when checkout fails.([DIS-1875](https://aspen-discovery.atlassian.net/browse/DIS-1875)) (*YL*)

### Koha Updates
- Fix an issue where multiple checked out items from the same record were not displayed separately in My Account checked out titles. ([DIS-1908](https://aspen-discovery.atlassian.net/browse/DIS-1908)) (*YL*)

### OverDrive Updates
- When loading reader name do not load full user object for improved performance. ([DIS-1799](https://aspen-discovery.atlassian.net/browse/DIS-1799)) (*MDN*)

### Polaris Updates
- Fix indexing stalling issue by making bibsToUpdate thread-safe, using a per0thread GroupedWorkIndexer to reduce lock contention. ([DIS-1895](https://aspen-discovery.atlassian.net/browse/DIS-1895)) (*YL*)

### Sierra Updates
- Make Phone and Work Phone fields configurable for Sierra. ([DIS-1809](https://aspen-discovery.atlassian.net/browse/DIS-1809)) (*MDN*)
- Prior to opting out of reading history in Sierra, clear the existing reading history. ([DIS-1791](https://aspen-discovery.atlassian.net/browse/DIS-1791)) (*MDN*)

### Side Load Updates
- Do not log that a URL has invalid 856 indicators unless no valid URLs are found. ([DIS-1831](https://aspen-discovery.atlassian.net/browse/DIS-1831)) (*MDN*)

### Symphony Updates
- Allow self checks to proceed if the user has a status of DELINQUENT, or if they have claims returned titles. ([DIS-1834](https://aspen-discovery.atlassian.net/browse/DIS-1834)) (*MDN*)

### Search Updates
- Fix multiple filter errors for lazy loading facets.  ([DIS-1808](https://aspen-discovery.atlassian.net/browse/DIS-1808)) (*TC*)

### Web Builder Updates
- Fix database error when uploading new images with empty Hero Slider integer fields. ([DIS-1837](https://aspen-discovery.atlassian.net/browse/DIS-1837)) (*LM*)

### Other Updates

- Do not redirect to SSO login on failed validation and redirect to proper page for login. ([DIS-1850](https://aspen-discovery.atlassian.net/browse/DIS-1850)) (*MDN*)
