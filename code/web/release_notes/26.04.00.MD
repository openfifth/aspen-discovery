### Search Updates
- Add library-level custom grouped work search specs configuration. ((DIS-1851)(https://aspen-discovery.atlassian.net/browse/DIS-1851)) (*TC*)

  - Enables performance tuning for large-scale deployments and site-specific search preferences
  - Preserves Aspen's excellent default search behavior while providing flexibility
  - See example file: `sites/default/conf/custom_grouped_work_search_specs.yaml.example`
