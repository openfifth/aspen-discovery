### Sierra Updates
- Add value "home_library_code" for pickup/home library location ([DIS-1966](https://aspen-discovery.atlassian.net/browse/DIS-1966)) (*KL*)
- Add ability to copy/paste information from the Sierra Admin Days Closed table into Aspen to populate Aspen Holidays ([DIS-281](https://aspen-discovery.atlassian.net/browse/DIS-281)) (*KL*)
  - Adds a new button in Library Systems and Locations to "Batch Update Holidays"
  - The user is prompted to paste information from Sierra into a text box - this can be comma separated values or values separated by spaces
  - The user can either update Libraries or Locations or Both
  - Locations do not have holiday functionality, so instead of adding location holidays we add weekly closure dates if the date in Sierra is a 3 letter day code
  - What Libraries/Locations get updated depends on the users "Administer Libraries" and "Administer Locations" permissions
