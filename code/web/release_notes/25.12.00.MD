### User List Updates
- Added persistent format filters for user lists that remain active across sessions and page navigation. (DIS-1583) (*LS*)
- CSV and RIS export functions now respect active format filters, only exporting filtered entries. (DIS-1583) (*LS*)
- Format filter counts now accurately count by grouped works instead of individual records/editions. (DIS-1583) (*LS*)
- Export operations now display error messages in modal dialogs when they fail. (DIS-1583) (*LS*)
- Fixed an issue where some list covers would fail to generate due to invalid records. (DIS-1583) (*LS*)

### Other Updates
- Fixed "Array to string conversion" warnings in RIS export when handling series information. (DIS-1583) (*LS*)
