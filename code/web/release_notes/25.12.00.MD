### Indexing Updates
- Resolved facet-selection gaps where translated location names didn't match the facet label, so scoped availability now remains consistent after renaming branches. (DIS-1571) (*LS*)
  - Owning-location names now come directly from each Location's "Facet Label" or "Display Name" instead of the deprecated "location" Translation Map.
  - If no location code matches, likely due to a reliance on regex matching, fall back to using the "location" Translation Map.
  - If it is available, the Facet Label for the Location will be used for single-location Library Systems.
