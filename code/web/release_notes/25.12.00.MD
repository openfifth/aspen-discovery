### Account Updates
- Respect Allow Freezing Holds option in Library Settings. Applies to CARL.X, Polaris, Sierra, Symphony. (DIS-1634) (*MDN*)

### Administration Updates
- If an error occurs while submitting a form, fill the form with values entered by the user so they can correct and resubmit. (DIS-1589) (*MDN*)
  - This works both when creating a new object and when editing an existing object. 
  - Any errors that cause the object to not save are reported to the user. 
- Add proper return types and clean up syntax errors in Object Editor and derived classes. (DIS-1625) (*MDN*)

### Code Cleanup
- Remove unused and unmaintained FOLIO, Horizon, Library.Solutions, and Millennium drivers. (DIS-1646) (*MDN*)

### Hold Updates
- Allow the Pickup location prompt to be hidden if only one pickup location is valid. (DIS-1624) (*MDN*)
- Allow administrators to disable changing pickup locations for Unavailable Holds. (DIS-1624) (*MDN*)

### Events Updates
- Focused the Springshare LibCal orphan cleanup so it only runs when at least one LibCal setting exists or when orphaned events still need removal after settings were deleted. (DIS-1546) (*LS*)

### Koha Updates
- Fixed Messaging Settings to honor the TalkingTech (itiva) phone transport and hide phone checkboxes on non-itiva-capable notices, matching Koha OPAC. (DIS-1643) (*LS*)

### Materials Requests Updates
- Fixed date filter in Manage Materials Requests to make the "To" date inclusive of the entire day, resolving issues where requests made on the current day were excluded from filter results. (DIS-1486) (*LS*)

### Placards Updates
- Implemented scoring-based placard matching algorithm to ensure the most relevant placard is selected when multiple triggers could apply. (DIS-1644) (*LS*)
  - Relevance scoring now prioritizes match specificity, addressing previous issues where broad triggers (e.g., "lead") were incorrectly favored over more specific ones (e.g., "union leader").
  - Improved partial-match logic for scenarios such as search: "union lead" with triggers "union leader" and "lead," correctly prioritizing the more relevant "union leader."
  - Added debug information about the matched placard when "Show Debugging Information" is enabled in the IP Address settings.
- Fixed an issue where links inside a placard could break formatting. Placard links now apply only to the image if the body has any links. (DIS-1636) (*LS*)
- Text inside placards now follows the formatting you choose in the editor instead of always appearing bold. (DIS-1636) (*LS*)

### Polaris Updates
- Fixed an issue where self registration required a hardcoded minimum age of 13 years; instead, it will use the "Minimum Age" field in Library System settings. (DIS-1621) (*LS*)

### Other Updates
- Fixed an issue where the Aspen Admin Local Administrator settings could not be edited because the Email Address field was read-only but also required. (DIS-1627) (*LS*)
- Added a "View Calendar" button to the Calendar Display Settings page for easy navigation to the calendar page. (DIS-1629) (*LS*)
- Included ISBN and UPC data in the CSV export of search results. (DIS-1379) (*LS*)
