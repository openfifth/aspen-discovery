## Aspen Discovery Updates

### Analytics Updates
- If a library has Cookie Consent turned off, log interactions with eContent. ([DIS-1709](https://aspen-discovery.atlassian.net/browse/DIS-1709)) (*MDN*)

### Bookcover Updates
- Properly generate Bookcovers for Lists and Series when there are fewer than 4 entries in the list or series. ([DIS-472](https://aspen-discovery.atlassian.net/browse/DIS-472)) (*MDN*)
- Properly encode Bookcover URL for Grouped Works when the format category has a space in it. ([DIS-472](https://aspen-discovery.atlassian.net/browse/DIS-472)) (*MDN*)

### Summon Updates
- Prevent Summon from logging errors for missing rssLinks to error.log. ([DIS-1707] (https://aspen-discovery.atlassian.net/browse/DIS-1707)) (*AB*)


### Community Engagement Updates
- Fix issue of the join leaderboard button displaying when the use leaderboards option is turned off. ([DIS-1731](https://aspen-discovery.atlassian.net/browse/DIS-1731)) (*AB*)

### eCommerce Report
- Fix Comprise API amount formatting to use plain decimal format without thousands separators for large donations. ([DIS-1789](https://aspen-discovery.atlassian.net/browse/DIS-1789))  (*YL*)
- Fix Certified Payments by Deluxe payments not being sent to ILS to clear fines after payment completion. ([DIS-1798](https://aspen-discovery.atlassian.net/browse/DIS-1798)) (*YL*)

### Evergreen Updates
- Fix an infinite loop during initial reading history import from Evergreen if patron authentication fails ([DIS-1696](https://aspen-discovery.atlassian.net/browse/DIS-1696)) (**GMC**)

### Hold Updates
- Now, if the allowMaxDaysToFreeze preference is enabled, the system calculates the maximum freeze period starting from the date the item was held (only Koha). ([DIS-1692](https://aspen-discovery.atlassian.net/browse/DIS-1692)) (*LM*)

### Hoopla Updates
- Fix Hoopla v2 so instant titles no longer duplicate once per member library; instant items now remain a single shared record with all applicable library scopes.([DIS-1812](https://aspen-discovery.atlassian.net/browse/DIS-1812)) (*YL*)

### Indexing Updates
- Ensure Grouped Work Indexing Diagnostics do not overflow column. ([DIS-1774](https://aspen-discovery.atlassian.net/browse/DIS-1774)) (*MDN*)
- Update 8xx series display to remove subfield $q and add subfield $n. ([DIS-1721](https://aspen-discovery.atlassian.net/browse/DIS-1721)) (*MDN*)

### OverDrive Updates
- Allow library to suppress OverDrive kindle format per scope. ([DIS-1714](https://aspen-discovery.atlassian.net/browse/DIS-1714)) (*YL*)

### Search Updates
- Make search results page more responsive by loading facets asynchronously. ([DIS-1808](https://aspen-discovery.atlassian.net/browse/DIS-1808)) (*LS*)
- Add library-level setting to toggle async facet loading. ([DIS-1808](https://aspen-discovery.atlassian.net/browse/DIS-1808)) (*TC*)

  - Provides admin control over search performance optimization while maintaining backward compatibility

### Sierra Updates
- Restore functionality to display Records without the check digit in the URL. ([DIS-1760](https://aspen-discovery.atlassian.net/browse/DIS-1760)) (*MDN*)

### Other Updates
- Remove "NintendoSwitch" format when "NintendoSwitch2" format is present. ([DIS-1725](https://aspen-discovery.atlassian.net/browse/DIS-1725)) (*YL*)
- Increase SHAREit search connection timeout for large collections. ([DIS-1782](https://aspen-discovery.atlassian.net/browse/DIS-1782)) (*YL*)
