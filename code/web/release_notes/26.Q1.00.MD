### Account Updates
- Show count of titles Ready For Pickup from eContent Sources on Account Home page. ([DIS-1845](https://aspen-discovery.atlassian.net/browse/DIS-1845)) (*MDN*)
- When caching Account Summary information preserve prior data rather than deleting from the database. ([DIS-1454](https://aspen-discovery.atlassian.net/browse/DIS-1454)) (*MDN*) 
  - Data will be marked as stale rather than clearing it. 
  - Data will automatically refresh after 5 minutes or when the data is stale. 
  - When placing holds (bib, item, or volume), the account summary will be automatically updated so it does not need to be refreshed.
  - When cancelling holds, the account summary will be automatically updated so it does not need to be refreshed.
- Improve caching of holds. ([DIS-1454](https://aspen-discovery.atlassian.net/browse/DIS-1454)) (*MDN*)
  - Mark data as stale rather than clearing it completely.
  - When reloading cached holds, the existing cache will be updated.
  - When clearing cache, only clear cache for the specific source rather than all sources.
  - When cancelling holds, the hold will be removed from the cache automatically so the source does not need to be queried.
  - When freezing and thawing holds, the hold will be updated in the cache automatically so the source does not need to be queried.
- Improve caching of checkouts. ([DIS-1454](https://aspen-discovery.atlassian.net/browse/DIS-1454)) (*MDN*)
  - Mark data as stale rather than clearing it completely.
  - When reloading cached checkouts, the existing cache will be updated.
  - When clearing cache, only clear cache for the specific source rather than all sources.
  - When returning eContent titles, the cache will be updated automatically. 
  - When checking out a title, the cache will be invalidated so it will refresh.
